# Product Vision

SWEOrchestrAI is a hybrid local/cloud AI Project Manager for software engineering workflows.

The system combines:

- cloud-based project management,
- local AI agent execution,
- repository-aware automation,
- design and diagram generation,
- MCP/tool integrations,
- approval-gated coding workflows,
- and multi-agent teams specialized by software delivery roles.

The main goal is to evolve from a simple "AI dev team manager" into a broader AI Project Manager that supports the full software delivery lifecycle.

## Lifecycle Scope

SWEOrchestrAI is intended to support:

1. Discovery
2. Requirements
3. Architecture
4. Design
5. Backlog
6. Implementation
7. Review
8. Documentation
9. Delivery

## Product Identity

> Cloud-managed software delivery. Locally executed AI engineering.

## MVP Focus

The MVP focuses on validating the local/cloud split:

- Cloud manages project state and visibility.
- Local service performs provider/tool/repository execution.
- Local front gives the user direct control over local runs and approvals.
- Cloud front gives project management visibility without directly performing AI coding.
