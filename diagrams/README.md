# Diagrams

This directory is reserved for SWEOrchestrAI architecture diagrams, exported visuals, and diagram source files.

## Guidelines

- Store exported images and source diagrams together where possible.
- Generated diagrams should include source prompts or generation metadata when useful.
- Prefer clear filenames that describe the diagram purpose.
- Keep diagrams aligned with the documentation in `docs/architecture`.
- Use this folder for portfolio-facing visuals and architecture references.

## Suggested Files

Examples:

- `mvp-architecture.png`
- `mvp-architecture.mmd`
- `local-cloud-boundary.png`
- `repository-dependencies.png`
- `communication-model.png`
